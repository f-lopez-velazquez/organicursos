# Seguridad

Si detectas una vulnerabilidad o un problema que pueda comprometer datos, integridad del sistema o ejecución local, evita publicarlo primero como issue abierto.

## Reporte responsable

Envía el reporte con la mayor precisión posible a:

- Correo: f.lopezvelazquez@ugto.mx
- Sitio: https://zolvek-mx.web.app

## Qué incluir

- descripción del riesgo
- impacto potencial
- pasos para reproducir
- sistema operativo afectado
- versión de la aplicación
- evidencia mínima necesaria para validar el problema

## Alcance

Se consideran especialmente sensibles:

- lectura o escritura fuera de rutas esperadas
- exposición de datos locales
- ejecución arbitraria
- escalamiento de privilegios
- empaquetado o sidecars alterados

Se intentará confirmar recepción, reproducir el reporte y coordinar una corrección antes de divulgarlo públicamente.
