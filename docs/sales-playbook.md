# Sales Playbook

## Paquete minimo para vender

- build firmado por plataforma
- landing page con posicionamiento local-first
- pagina de privacidad resumida
- FAQ de licencias offline
- flujo de soporte con respaldo e importacion

## Ediciones sugeridas

- Community: base local-first, biblioteca, reproduccion, notas, busqueda
- Professional: soporte prioritario, enriquecimiento opt-in, playbooks de operacion y licenciamiento individual
- Team: despliegue por volumen, licencias multi-equipo y soporte coordinado

## Flujo comercial sugerido

1. Preventa o demo.
2. Emision de token firmado.
3. Cliente activa desde la pantalla `Licencia`.
4. Cliente valida edition, expiracion y soporte.
5. Soporte usa la pantalla `Soporte` para diagnostico inicial.

## Contacto comercial de referencia

- sales@organicursos.app
- https://organicursos.app/pricing

## Checklist de preventa

- confirmar plataforma objetivo
- confirmar volumen de bibliotecas
- confirmar si se necesita despliegue administrado
- confirmar si se requiere politica de renovacion anual

## Checklist de postventa

- enviar token firmado
- enviar guia de activacion
- enviar guia de respaldo
- registrar version entregada y canal de release
